Project status

Status and changelog

FOCUS is in active development. This page gives people a plain-language view of what is working now and what still needs to happen before production deployment.

Current status

The app is not production-ready yet. Local development workflows, automated tests, and core production-management features are in place, but production deployment settings and external authentication provider integrations are still unfinished.

Recent implementation work

  • Passkey sign-in, backup keys, and account safety guardrails.
  • Profiles with bio, availability, and project visibility controls.
  • Production groups with invitations, role management, member removal, and leave-group safeguards.
  • Projects with assignment tracking, status updates, resources, notes, archive, restore, export, and deletion.
  • Project and group notifications, notification filters, and site-wide notification announcements.

Production readiness still needed

  • Provider integrations for real sign-in anchors beyond local development.
  • Environment-based secrets, allowed hosts, secure cookies, and deployment documentation.
  • Backups, retention decisions, and operational recovery notes.
  • Automated accessibility checks and broader manual assistive technology testing.
  • Contributor workflow notes for issue triage, pull requests, reviews, and releases.
